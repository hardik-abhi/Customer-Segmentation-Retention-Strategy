# 🎯 Customer Segmentation & Retention Strategy

> RFM-based customer segmentation on 800K+ real e-commerce 
> transactions — identifying revenue drivers and delivering 
> a data-backed retention strategy.

## 🔍 Key Findings
- 22% of customers (Champions) drive 68% of revenue (£12.1M)
- 615 At-Risk customers have £1.5M revenue at stake
- 1,275 Lost customers haven't purchased in 468 days
- Classic Pareto Principle confirmed in real transaction data

## 🛠️ Tools
Python | SQL (pandasql) | Power BI | Excel | PowerPoint

## 📊 Segments Identified
| Segment | Customers | Revenue |
|---|---|---|
| Champions | 1,300 | £12.1M |
| Loyal Customers | 1,134 | £2.6M |
| At Risk | 615 | £1.5M |
| Needs Attention | 1,111 | £741K |
| Potential Loyalists | 443 | £394K |
| Lost | 1,275 | £327K |

## 💡 Strategic Recommendations
1. VIP Loyalty Program for Champions → +£1.8M
2. Win-back campaign for At-Risk → +£464K
3. Lost customer reactivation → +£49K
4. Total projected impact: £2.3M+
